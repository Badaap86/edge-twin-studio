# Pilot Acceptance Criteria

A paid EdgeTwin pilot is considered successful when the customer receives a controlled evidence bundle that clearly answers:

1. Is the provided data usable for the selected pilot goal?
2. What data quality issues exist?
3. Which use-case or pack is recommended?
4. What assumptions and limitations apply?
5. What risks block stronger claims?
6. What next data or validation steps are required?
7. Is this ready for a controlled pilot, or should the customer first improve data collection?

## Not included unless explicitly agreed

- Production deployment
- Accuracy guarantee
- Legal/compliance certification
- Safety certification
- Full system integration
- Long-term SaaS hosting
- Replacement of human review
